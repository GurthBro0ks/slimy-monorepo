## How to Run Things

### Locally

From the monorepo root:

```bash
# Enable pnpm v10
corepack enable && corepack prepare pnpm@10.22.0 --activate

# Install dependencies
pnpm install

# Generate Prisma client
pnpm --filter @slimy/web run db:generate

# Build all apps
pnpm build

# Run tests
pnpm --filter @slimy/web test

# Start dev server
pnpm --filter @slimy/web run dev
```

See [../dev/README.md](../dev/README.md) for more development workflows.

### On NUC2 (Production)

See the comprehensive deployment guide at [../infra/DOCKER_DEPLOYMENT.md](../infra/DOCKER_DEPLOYMENT.md).

**Quick NUC2 Deploy:**

```bash
# On NUC2 via SSH
cd /opt/slimy/slimy-monorepo
git pull

# Build locally first (recommended)
corepack enable && corepack prepare pnpm@10.22.0 --activate
pnpm install
pnpm --filter @slimy/web run db:generate
pnpm build

# Deploy with Docker
cd infra/docker
docker compose -f docker-compose.slimy-nuc2.yml build
docker compose -f docker-compose.slimy-nuc2.yml up -d
```
